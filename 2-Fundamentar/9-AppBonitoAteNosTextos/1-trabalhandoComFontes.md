# Trabalhando com fontes

Tipografia transmite mensagem

- negrito
- tamanho
- estilo

## Basic Font Properties

- font-family
- font-weight
- font-style
- font-size

## Font Family

- Tipo de fonte de um elemento
- Lista de fontes e ordem de prioridade
- Inclui _fallback_ font

```css
p {
  font-family: 'Times New Roman', Times, serif;
}
```

- serif
- sans

## Font weight

- Peso da fonte

```css
p {
  font-weight: bold;
}
```

## Font style

- O estilo da fonte

```css
span {
  font-style: italic;
}
```

## Font size

- O tamanho da fonte

```css
span {
  font-size: 18px;
}
```
