# Propriedades para os itens

- flex-basis
- flex-grow
- flex-shrink
- flex
- order

# flex-grow

- O crescimentos do item dentro do container em relação aos espaços vazios.

# flex-shrink

- O encolher do item dentro do container

# flex

- shorthand
- flex-grow flex-shrink flex-basis
- podem ter 1, 2 ou 3 valores
